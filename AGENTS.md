# Smart Caregiver Repository Guide

## Repository Structure
- **Server**: FastAPI backend located in the `server` directory.
- **Mobile**: Flutter application located in the `mobile` directory.

## Key Commands
- **Server**: Refer to `server/README.md` for server-specific commands and setup.
- **Mobile**: Refer to `mobile/README.md` for mobile-specific commands and setup.

## Workflow
1. **Development**: Work on features or fixes in the respective directories (`server` or `mobile`).
2. **Testing**: Ensure to test changes thoroughly within their respective environments

## Important Notes
- Always refer to the specific `README.md` files in the `server` and `mobile` directories for detailed setup and usage instructions
